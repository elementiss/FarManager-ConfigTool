# FarManager ConfigTool

A tool for manipulation FarManager settings export-import files


Far manager позволяет экспортировать настройки в xml-файл с помощью команды `far.exe /export settings.farconfig` и затем импортировать с помощью команды `far.exe /import settings.farconfig`. [Обсуждение](https://forum.farmanager.com/viewtopic.php?t=7009)

Недостатки:

- Нет возможности раздельного экспорта/импорта различных категорий настроек. [0002656](https://bugs.farmanager.com/view.php?id=2656)
- При импорте ассоциаций все текущие ассоциации удаляются (если есть узел `<associations>` в xml). Таким образом, невозможно добавить одну или несколько ассоциаций с помощью импорта, не перезаписывая все уже имеющиеся ассоциации. [Форум](https://forum.farmanager.com/viewtopic.php?t=9121)
- В файл экспорта, в частности, включается история, которая может содержать конфиденциальные данные

ConfigTool решает эти проблемы, предоставляя:

- Возможность выделить одну или несколько категорий настроек из общего файла экспорта, для дальнейшего раздельного импорта ассоцияаций, групп раскрасок и т.д. 
- Возможность выделить настройки конкретного плагина в отдельный файл 
- Удаление одной или нескольких категорий настроек из файла экспорта
- Сравнение двух файлов экспорта и формирование отчета
- Наложение патча с отдельными настройками на базовый файл настроек, с целью дальнейшего импорта
- Получение списка установленных плагинов, включая guid и описание
- Просмотр и редактирование настроек в удобной форме (в работе)
- Командная строка для выполнения операций извлечения, удаления, слияния и сравнения
