# FarManager ConfigTool

A tool for working with FarManager settings export-import files

Far позволяет экспортировать настройки в xml-файл с помощью команды `far.exe /export settings.farconfig` и затем импортировать с помощью команды `far.exe /import settings.farconfig`. [Обсуждение](https://forum.farmanager.com/viewtopic.php?t=7009)

Недостатки:

- Нет возможности раздельного экспорта различных категорий настроек. [0002656](https://bugs.farmanager.com/view.php?id=2656)
- При импорте ассоциаций все текущие ассоциации удаляются (если есть узел associations в xml). Таким образом, невозможно добавить одну или несколько ассоциаций с помощью импорта, не перезаписывая все уже имеющиеся ассоциации. [Форум](https://forum.farmanager.com/viewtopic.php?t=9121)
- В файл экспорта, в частности, включется история, которая может содержать конфиденциальные данные

ConfigTool решает эти проблемы, предоставляя:

- Возможность выделить один или несколько категорий из общего файла экспорта, для дальнейшего раздельного импорта ассоцияаций, групп раскрасок и т.д. 
- Возможность выделить настройки конкретного плагина в отдельный файл 
- Удаление одной или нескольких категорий из файла экспорта
- Сравнение двух файлов экспорта и формирование отчета
- Слияние двух файлов экспорта с целью дальнейшего импорта, с выбором стратегии:
  - полная замена (как сейчас в ассоциациях)
  - объединение: замена значений существующих настроек по ключу и добавление новых
  - замена по ключу: затереть только измененные параметры 
  - игнорирование раздела (не импортировать вообще)
  - удаление отдельных настроек (например, одной ассоциации) - не планируется. Только полная замена всех настроек категории
- Просмотр и редактирование настроек в удобной форме
- Командная строка для выполнения операций извлечения, удаления, слияния и сравнения
